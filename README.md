# Typing Time

Typing Time is an educational website that helps beginners learn keyboard skills.

## Purpose
To improve typing speed and accuracy through interactive typing exercises.

## Features
- Learn home row keys
- Practice letters, words, and sentences
- Instant feedback
- Accuracy tracking
- Multiple levels

## Technologies Used
- HTML
- CSS
- JavaScript

## Target Users
Primary school students and beginners learning to type.

## Future Improvements
- Timer system
- Keyboard visualisation
- Sound effects
- Leaderboard
